# WDTT Panel Android

**Текущая версия: 0.1.0**

Нативное Android-приложение на Kotlin + Jetpack Compose для управления несколькими серверами WDTT Control Panel.

## Возможности

- список серверов с мгновенным переключением активного сервера;
- вход по паролю панели через `/api/v1/auth/login`;
- защищённое локальное хранилище серверов, паролей и bearer-токенов через Android Keystore;
- поддержка self-signed TLS для серверов, где ещё нет публичного сертификата;
- обзор сервера: активные подключения, пользователи, устройства, трафик, CPU/RAM/диск, состояние сервиса;
- пользователи: список, создание, включение/отключение, отвязка устройства, сброс трафика, удаление;
- логи WDTT;
- быстрые команды сервиса: start, restart, stop.

## Требования

- Android 8.0+.
- WDTT Control Panel 0.11.8+ с включённым `/api/v1`.
- Для обычного режима нужен HTTPS-сертификат, доверенный Android. Для self-signed включите переключатель `Self-signed TLS` в карточке сервера.

## Сборка

```bash
./gradlew assembleDebug
```

Debug APK появится в `app/build/outputs/apk/debug/`.

## Ревизии

Версия приложения хранится в `VERSION` и `app/build.gradle.kts` (`versionName` / `versionCode`). При каждом пользовательском изменении поднимайте версию, обновляйте `CHANGELOG.md` и отправляйте изменения в `main`.
