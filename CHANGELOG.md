# Changelog

## 0.4.2

- Добавлен выбор VK-хешей из библиотеки в формах создания, редактирования и массового редактирования пользователей.
- Добавлено массовое редактирование выбранных пользователей: общий VK-хеш, порты и срок действия.
- Поле VK-хеша теперь поддерживает быстрое добавление и снятие выбранных библиотечных хешей до лимита WDTT.

## 0.4.1

- Нажатие на VK-хеш в библиотеке теперь копирует его, а удаление вынесено в отдельную кнопку.
- Добавлено копирование VK-хешей и портов в карточках пользователей и qWDTT-профилей.
- Добавлено редактирование пользователя из раздела «Люди» и из связанной карточки qWDTT-профиля.

## 0.4.0

- Добавлена вкладка настроек с управлением Telegram-ботом WDTT: включение, Admin ID, Bot Token и тестовое сообщение.
- Добавлен импорт и экспорт библиотеки VK-хешей JSON для переноса между серверами.
- GitHub Actions теперь создаёт GitHub Release с подписанным APK после пуша в `main`.
- Минимальная версия панели для полного набора функций поднята до WDTT Control Panel 0.11.12.

## 0.3.0

- Добавлена вкладка qWDTT-профилей с карточками пользователей, QR-кодами, экспортом ссылок, JSON и `.qwdtt`.
- Добавлены импорт из буфера/текста для `wdtt://`, `qwdtt://`, JSON и Base64.
- Добавлено управление библиотекой VK-хешей, qWDTT-подписка и фоновые проверки серверов с уведомлениями.
- Добавлены latency/status-индикаторы серверов и расширенные фильтры пользовательских представлений.

## 0.2.0

- Полностью обновлён интерфейс Jetpack Compose: современный обзор сервера, карточки состояния, ресурсные индикаторы, тёмная тема и иконки для направлений трафика.
- Добавлены поиск, фильтры, сортировка и массовые действия для пользователей.
- Добавлены авто-создание, пакетное создание пользователей и окно копирования `wdtt://` ссылок.
- Расширен экран логов: источники WDTT/панель/Nginx/установщик, лимиты строк, фильтр и копирование.
- Добавлена release-подпись через локальный `signing.properties` и GitHub Actions workflow для signed APK.

## 0.1.0

- Первый нативный Android-клиент на Kotlin + Jetpack Compose.
- Добавлены профили серверов, мгновенное переключение, парольный вход и bearer-токены.
- Добавлены обзор, пользователи, логи и основные команды сервиса.
- Пароли и токены хранятся в зашифрованном виде через Android Keystore.
